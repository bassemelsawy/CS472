    jQuery functions are actions that can be used on a jQuery object. When the jQuery function is invoked with a CSS selector, it will return a jQuery Object wrapping all the elements that match the selector. The jQuery object has properties like length and methods on it like ready(), click() etc. The jQuery object just wraps the DOM elements to make interactivity and DOM manipulation easier.
