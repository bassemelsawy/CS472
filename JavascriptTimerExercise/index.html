What is the difference between the following 2 statements? 
                setTimeout(booyah, 2000);
                setTimeout(booyah(), 2000);
setTimeout(booyah, 2000) calls booyah after a delay of 2 seconds whereas setTimeout(booyah(), 2000) calls the function booyah immediately.

What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?
Unobtrusive Javascript is a way of using JS code in the front end such that the HTML structure and behavior are separated. Unobtrusive Javascript is used to:
1. Separation of functionality/behavior from a Web page's structure/content and presentation
2. To avoid the problems of traditional JavaScript programming and create more modular codes
